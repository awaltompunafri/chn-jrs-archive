# chn-jrs-archive

本仓库用于归档和发布多个独立 HTML 页面。

## 项目简介

chn-jrs-archive 是一个静态 HTML 页面归档仓库。这里存放的页面以独立文件形式存在，不依赖于特定域名或网站系统。每个页面均可直接通过浏览器打开，适合作为资料存档或离线展示用途。

## 目录说明

```
/
├── pages/          # 存放归档的 HTML 页面文件
├── assets/         # 页面所需的资源文件（样式、脚本、图片等）
├── index.html      # 仓库首页或导航页面（可选）
└── README.md       # 本说明文件
```

- `pages/`：主要归档目录，每个 HTML 文件为一个独立页面。
- `assets/`：包含页面引用的外部资源，按类型或用途组织子目录。
- `index.html`：如存在，可作为仓库页面总览或入口。

## 页面归档说明

- 所有页面均为独立的 HTML 文件，可直接在浏览器中运行。
- 页面之间无强制依赖关系，可单独访问或部署。
- 归档内容可能包含静态文本、图表、交互组件等，具体以页面实际内容为准。
- 页面命名遵循简洁、可辨识的原则，文件名可反映页面主题或用途。

## 维护说明

- 本仓库由维护者不定期更新，添加或修正归档页面。
- 欢迎通过 Issue 或 Pull Request 提出建议或提交内容。
- 提交新页面时，请确保文件为有效 HTML，并放置于 `pages/` 目录下。
- 资源文件请放置于 `assets/` 中对应子目录，避免路径混淆。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接打开 `pages/` 下的 HTML 文件，或通过 `index.html` 导航浏览。
3. 也可将仓库内容部署至任意静态托管服务，通过 URL 访问各个页面。
